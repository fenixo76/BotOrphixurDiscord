# Manuel

## Remplire le fichier config.json

```json
{
    "clientId": "ID bot",
    "guildId": "ID serveur Discord",
    "Token": "Token bot",
    "channelId": "ID channel"
}
```

## Installer et lancer le bot

### Avec npm

```bash
npm i
npm start
```

### Avec node

```bash
npm i
npm run build
node .
```
